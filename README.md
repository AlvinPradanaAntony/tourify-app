# Tourify App Backend

Backend apps for Tourify App

## Setup

Note : Required NodeJS version : >= 18.0.0

- Install yarn
  ```bash
  npm i -g yarn
  ```

* Check yarn version
  ```bash
  yarn -v
  ```
* Install dependencies
  ```bash
  yarn
  ```
* Copy .env.example to .env
  ```bash
  cp .env.example .env
  ```
