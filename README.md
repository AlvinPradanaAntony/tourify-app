# Tourify App Backend

Backend apps for Tourify App

## Setup

Note : Required NodeJS version : >= 18.0.0

* Install dependencies
  ```bash
  npm install
  ```
* Copy .env.example to .env
  ```bash
  cp .env.example .env
  ```
* Migrate all tables
  ```bash
  npm run migrate
  ```
* Seed all data
  ```bash
  npm run seed
  ```
